# 熊の字印の漢検準一級暗記帳

Windows/Androidで動作するQt/C++/QML製 漢検準一級暗記用アプリ  
（準一級に拘る必要が無い気がします）

![スクリーンショット](image/screenshot.png)

## 概要

漢字検定準一級を受験したので、15年くらい前に受験した時に落ちた時の雪辱をギリギリで果たせた気がするので、記念に作りました、というつもりではなくて、ホントは試験本番前に（漢検とQtの）勉強用に作るつもりだったのですが、気付いたら試験自体が先に終わってたので、このアプリが後付けになりました。

手元にノートを準備して書き取りしながら使うのを想定してます。試験本番前に完成してたら意外とちゃんと勉強に使えたはずです。

よく考えると漢検一級受験にも流用出来そうな気がしますが、勉強量が膨大すぎるので流用されることはない気がします（`咀嚼`とか`赫奕`とか漢字で書けるようになる機会が得られないのは心残りではありますが）

## 解説

- 出題用CSVを読み込むと出題されます。とりあえず添付のsource.csvを読み込んでみてください。
- csvファイルを編集することで問題を追加したりいろいろできます。書式は添付のsource.csvに倣って下さい。
- 正答数・誤答数をファイルに書き込もうとする都合上、変なファイルを読み込むとそのファイルのデータが吹っ飛ぶ可能性もありますのでご注意下さい。
- いちおうWindows/Androidのクロスプラットフォームですが、UIが完全にWindowsを念頭に置いており、Androidスマホでは画面が圧縮されすぎて操作無理ゲーなので、Windowsで使って下さい。
- source.csvの元ネタは私が勉強中に分からなかった問題です。（source.csvを作るところまでは試験本番前にやりました）

## インストール
- Windows版はzipを展開して、`flashcards.exe`を実行してください。
- Android版はzipを展開して、中のflashcards.apkをインストールしてください。「不明なアプリのインストールを許可」が必要な場合があります。
- あといずれもzipの中にsource.csvが添付されてます。
